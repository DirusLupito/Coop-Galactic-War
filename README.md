# Coop-Galactic-War
Contains Planetary Annihilation UI and Server Script modifications aimed at supporting cooperative galactic war play.

To install, either copy ui and server-script into your install to overwrite everything, or do something much more efficient and run the python script [generate_mod.py](generate_mod.py) to generate a folder called media, then drag and drop that folder into your steam install to overwrite only those files which have been modified to allow for cooperative play.
